Setup:
1. npm install
2. npm start

Run Game:
1. create grid, allowed inputs 4-40 for size of grid (length x width)
2. pick pattern from lists
3. set pattern to render generation 0 on grid
4. start/ pause
5. can modify speed of generation tick by 100ths of a second,
    10 = 1 sec, 1= .1 sec, 100 = 10 sec
6. reset to change parameters
